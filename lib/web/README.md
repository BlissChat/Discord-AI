# Discord AI Assistant

## Quick start
1. Copy files into a fresh repo or Replit project.
2. `cp .env.example .env` and fill the variables.
3. `npm install`
4. `node index.js`

## Development
- Use `GUILD_ID` in `.env` to register commands instantly to a test server.
- Plugins go into `/plugins`.
- Web dashboard: `node web/dashboard.js`.

## Notes
- Watch OpenAI usage & tokens.
- Add your own admin UIs, more plugins, or swap to persistent DB for scale.
